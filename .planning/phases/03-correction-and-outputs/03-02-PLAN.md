  <action>
    1. Update pyproject.toml: add "matplotlib>=3.7" and "adjustText>=1.0" to dependencies (if not already present from Task 1 of Plan 01).

    2. Write tests/test_plotter.py (smoke tests):
       - Create a synthetic corrected DataFrame (~30 rows, 3 domains, mix of omnibus and one_vs_rest, some significant some not, some NaN p_values).
       - test_ovr_manhattan_creates_png: call manhattan_plot() for a specific cluster_label, check output PNG exists, file size > 0, and DPI is 300 (read back with PIL.Image or matplotlib.image).
       - test_omnibus_manhattan_creates_png: call omnibus_plot(), same checks.
       - test_ovr_uses_triangle_markers: (optional, check that function completes without error on data with both positive and negative effect sizes).
       - test_no_significant_hits_no_crash: call manhattan_plot with data where no results pass FDR -- should still produce a valid plot (no labels, but no crash).
       - Use tmp_path fixture for output directory.

    3. Write src/abcd_phewas/plotter.py:

       Helper: _build_x_positions(df_plot, domain_config) -> (df_plot_with_x_pos, domain_centers):
       - Sort by domain order (from domain_config YAML ordering), then alphabetically within domain.
       - Assign sequential x positions with gap=5 between domains.
       - Compute domain center x positions for x-axis labels.
       - Return augmented DataFrame and domain_centers dict.

       Helper: _add_threshold_lines(ax, df_plot, n_tests):
       - Bonferroni line: ax.axhline(-log10(0.05 / n_tests), ...) in red dashed.
       - FDR line: find largest p_value where fdr_q_global <= 0.05. If exists, draw blue dashed line at -log10(that p_value). If no significant hits, omit FDR line.
       - Add legend entries for both lines.

       Helper: _add_labels(ax, df_plot, x_col, y_col, rename_map=None):
       - Label selection: Bonferroni-significant hits get labels. If more than 20 Bonferroni-significant, take top 20 by p_value. If fewer than 5 Bonferroni-significant, also add FDR-significant up to 15 total.
       - Use rename_map (optional dict) to translate variable names to display labels.
       - Create text annotations, then call adjust_text() with arrowprops=dict(arrowstyle='->', color='gray', lw=0.5).
       - MUST be called LAST, after all axis limits and elements are set.

       Function: manhattan_plot(df, cluster_label, domain_config, output_path, rename_map=None):
       - Filter df to comparison_type == "one_vs_rest" and cluster_label == cluster_label.
       - Compute neg_log_p = -log10(p_value). Handle NaN/0 p-values (clip to 1e-300 before log).
       - Build x positions with _build_x_positions.
       - Create figure (16, 6 inches).
       - Scatter with directional markers: split into positive ('^') and negative ('v') effect_size. Color by domain hex color from domain_map. Two separate ax.scatter calls (matplotlib limitation).
       - Add alternating light gray background bands per domain for visual grouping.
       - Add threshold lines via _add_threshold_lines. n_tests = total OVR tests in the full DataFrame (not just this cluster -- use len(df[df.comparison_type == 'one_vs_rest'])).
       - Set x-axis: domain center ticks with domain names, rotated 30 degrees.
       - Set y-axis: "-log10(p-value)".
       - Title: "PheWAS: Cluster {cluster_label} vs. Rest".
       - Add labels via _add_labels.
       - savefig at 300 DPI with bbox_inches='tight'.
       - plt.close(fig).

       Function: omnibus_plot(df, domain_config, output_path, rename_map=None):
       - Filter df to comparison_type == "omnibus".
       - Same structure as manhattan_plot but:
         - Use circular markers ('o') instead of directional triangles (omnibus has no direction).
         - Single scatter call (no marker splitting needed).
         - n_tests = number of omnibus tests.
         - Title: "PheWAS: Omnibus Test (All Clusters)".

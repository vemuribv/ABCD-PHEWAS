    - test_non_skewed_zscore: Non-skewed array (normal distribution) -> z-scored directly (no winsorization). Output has mean ~0, std ~1.
    - test_transformation_log: Each processed column produces a log dict with keys: variable, skew_initial, winsorized, skew_post_winsor, int_applied, z_scored. Values match the actual path taken.

    - test_binary_passthrough: Binary column values are identical before and after preprocess_dataframe call.
    - test_winsorize_mean_sd: Array with outliers at +/- 5 SD are clipped to mean +/- 3 SD bounds.
    - test_rank_based_int: Known input ranks produce correct normal quantile values (compare to scipy.stats.norm.ppf).
    - test_nan_handling: Arrays with NaN values -- NaN positions preserved through all transformations.
  </behavior>
  <action>
    1. Create src/abcd_phewas/preprocessor.py with:
       - Import scipy.stats (skew, rankdata, norm), numpy. Import VarType from type_detector.
       - SKEW_THRESHOLD = 1.96 (default, overridable from config)
       - _scipy_skew(arr: np.ndarray) -> float: Compute skewness using scipy.stats.skew(bias=True) on non-NaN values. Return 0.0 if fewer than 3 non-NaN values.
       - winsorize_mean_sd(arr: np.ndarray, n_sd: float = 3.0) -> np.ndarray: Compute mean and std (ddof=1) on non-NaN values, clip to [mean - n_sd*std, mean + n_sd*std]. Preserve NaN positions. This is MEAN-BASED, not percentile-based (matching R's DescTools::Winsorize with minval/maxval).
       - rank_based_int(arr: np.ndarray) -> np.ndarray: For non-NaN values, compute average ranks via rankdata, then norm.ppf((ranks - 0.5) / n). Preserve NaN positions. Matches R's qnorm((rank(x, na.last="keep")-0.5)/sum(!is.na(x))).
       - z_score(arr: np.ndarray) -> np.ndarray: (arr - mean) / std with ddof=1. If std == 0, return zeros. Preserve NaN.
       - preprocess_continuous_column(arr: np.ndarray, col_name: str, skew_threshold: float = 1.96, winsor_n_sd: float = 3.0) -> tuple[np.ndarray, dict]: Two-pass pipeline. Returns (transformed_array, log_dict). Log dict has keys: variable, skew_initial, winsorized, skew_post_winsor, int_applied, z_scored.
       - preprocess_dataframe(df: pd.DataFrame, type_map: dict[str, VarType], pheno_cols: list[str], config: PipelineConfig) -> tuple[pd.DataFrame, pd.DataFrame]: Process only CONTINUOUS columns through preprocess_continuous_column. Leave ORDINAL, BINARY, CATEGORICAL untouched. Return (modified_df, transformation_log_df). Transformation log is a DataFrame with one row per continuous variable.

    - Use ddof=1 for std in both winsorize and z_score -- matches R's default scale()
    - NaN handling: Use np.nanmean, np.nanstd, and mask non-NaN values for rankdata

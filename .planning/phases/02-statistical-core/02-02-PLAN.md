       - `run_kruskal_wallis(variable, groups_dict, var_type) -> dict` — KW across all groups, effect_size=epsilon_squared, effect_size_type="epsilon_squared". groups_dict = {cluster_label: array_of_values}. Drop NaN from each group before test. Check all groups have >= 2 obs.

       - `run_chi_square_omnibus(variable, contingency_table, var_type) -> dict` — chi-square on KxL table. effect_size=cramers_v. No sparse fallback for omnibus -- justification: omnibus contingency tables aggregate ALL K clusters (K=2 to 8) into rows, so expected cell counts are distributed across the full sample (n=5556). With minimum cluster sizes of ~50 subjects and binary/categorical variables having 2-10 levels, expected cell frequencies virtually never fall below 5. The sparse-cell problem is specific to one-vs-rest 2xL tables where a single small cluster is isolated. Applying Monte Carlo simulation to KxL omnibus tables would add unnecessary computation with no practical benefit. Per CONTEXT.md, the fallback chain is specified only for one-vs-rest comparisons.
       - `run_chi_square_pairwise(variable, target_values, rest_values, cluster_label, var_type) -> dict` — chi-square on 2xL table with sparse fallback chain: (a) compute expected freqs, (b) if any expected < 5 AND table is 2x2: Fisher's exact, (c) if any expected < 5 AND table > 2x2: monte_carlo_chi_square, (d) else: standard chi-square. effect_size=cramers_v. CI via bootstrap_ci.

  <name>Task 2: Implement effect_sizes.py (GREEN)</name>
  <files>src/abcd_phewas/effect_sizes.py</files>
  <action>
    Create src/abcd_phewas/effect_sizes.py with these functions:

    1. `cohens_d(group1: np.ndarray, group2: np.ndarray) -> float`
       - Pooled SD formula: sqrt(((n1-1)*var1 + (n2-1)*var2) / (n1+n2-2))
       - Return 0.0 if pooled_sd == 0 (both groups constant)
       - Direction: mean(group1) - mean(group2) (target - rest)

    2. `rank_biserial(u_statistic: float, n1: int, n2: int) -> float`
       - Formula: 1 - (2 * U) / (n1 * n2)
       - Returns value in [-1, 1]

    3. `cramers_v(contingency_table: np.ndarray) -> float`
       - Uses chi2_contingency(correction=False) internally
       - sqrt(chi2 / (n * min(rows-1, cols-1)))
       - Returns 0.0 if min_dim == 0 or n == 0

    4. `epsilon_squared(h_statistic: float, n_total: int) -> float`
       - Formula: H / (n - 1)
       - Returns 0.0 if n_total <= 1

    5. `monte_carlo_chi_square(observed: np.ndarray, n_replicates: int = 10_000, rng: np.random.Generator | None = None) -> float`
       - Compute observed chi2 via chi2_contingency
       - Generate random tables with same marginals using multinomial sampling: for each replicate, create a random table by distributing n total observations into cells proportional to the expected frequencies under independence, then compute chi2
       - Return (count_ge + 1) / (n_replicates + 1) for conservative p-value
       - Accept rng parameter for reproducibility

    6. `bootstrap_ci(statistic_fn: Callable, data_args: tuple[np.ndarray, ...], n_resamples: int = 2000, confidence_level: float = 0.95, random_state: int | None = None) -> tuple[float, float]`
       - Wrapper around scipy.stats.bootstrap with method="percentile" (not BCa -- BCa fails on degenerate data)
       - Catch exceptions and DegenerateDataWarning -> return (nan, nan)
       - The statistic_fn must accept (*samples, axis) signature per scipy bootstrap API

    All functions should have docstrings with parameter descriptions and return types.
    Use `from __future__ import annotations` for union type hints.
  </action>
